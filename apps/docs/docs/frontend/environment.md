## Frontend environment variables

The frontend supports following environmental variables:

| Variable             | Purpose                                            | Notes                                                    |
| -------------------- | -------------------------------------------------- | -------------------------------------------------------- |
| `PUBLIC_BACKEND_API` | Public URL of the backend API used by the frontend | Include scheme and path (e.g., `https://chithi.dev/api`) |

!!! Tip

    Make sure the backend has CORS and proxy match
